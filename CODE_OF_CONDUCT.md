# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct.

In short: treat others with respect, assume good intent, and keep discussion
focused on the work. Behavior that makes the project unwelcoming is not
acceptable.

## Scope

This Code of Conduct applies to all project spaces, including the GitHub
repository (issues, pull requests, discussions), any official communication
channels, and public spaces when an individual is representing the project.

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
please report it privately to the maintainers by opening a
[GitHub Security Advisory](https://github.com/tkuchel/foldersync/security/advisories/new)
with the subject "Code of Conduct report". All reports will be reviewed
promptly and kept confidential.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of
Conduct, and will take appropriate and fair corrective action in response to
any behavior they deem inappropriate. Maintainers have the right to remove,
edit, or reject comments, commits, code, issues, and other contributions that
are not aligned with this Code of Conduct.

The full enforcement guidelines used by this project are documented in the
[Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines).
